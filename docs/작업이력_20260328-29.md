# 작업 이력 (2026-03-28 ~ 03-29)

## UI/UX 개선

### 입력폼
- "시뮬레이션 종료 나이" → "몇 세까지 계산할까요?"로 변경
- 부채 & 비정기 지출 타이틀 앞 이모지(🔻) 제거
- 기타 자산(custom) 선택 시 이름 입력 필드 제거 — 셀렉트로 바로 잔액 입력
- 62세 은퇴자 프리셋: retirementStartAge 60→62 변경, 설명문 수정
- 삭제 아이콘 색상 진하게 (#ccc → #aaa)
- 셀렉트박스 다크그레이(#4b5563) 배경
- 카드(asset-card) 내부 인풋 배경 흰색
- 퇴직연금/개인연금/추가자산 삭제 버튼: 휴지통 아이콘(Trash2), hover 시 빨간 배경
- 유형 선택: 버튼 → 셀렉트박스로 통일 (IRP/DC/DB, 연금저축/연금보험, ISA/해외직투 등)
- 추가 버튼: dashed border "계좌 추가" 형태로 통일
- 은퇴시점/적립식 토글: pill 스위치 스타일 (회색 배경 + 흰색 활성)
- "내 정보 직접 입력하기" 버튼: 보더 없음, fontWeight normal, PenLine 아이콘
- 모든 버튼에 hover 스타일 추가 (삭제/추가/계산/초기화/셀렉트 등)
- PC 기본정보 라벨 고정 너비(140px)로 인풋 시작점 정렬
- 계산하기 버튼 disabled 시 boxShadow/펄스 애니메이션 제거

### 프리셋(연령별 탈출 시나리오)
- 설명문 대화체로 변경 (br 없이 자연스럽게 이어짐)
- "나와 비슷한 상황 보기" → "연령별 탈출 시나리오"
- 파란 설명 박스: 위쪽 삼각형 화살표(^) 추가, 닫으면 완전히 사라짐
- 프리셋 재선택 시 설명 박스 다시 열림

### 툴팁
- 검정 배경 → 흰색 배경 + 그림자로 전면 변경 (은퇴계산기 + ActionPlan 모두)
- 위쪽에 표시 + 하단 뾰족 화살표
- 상단 공간 부족 시 자동으로 아래쪽 표시 (ActionPlan PlanTooltip)
- "희망 은퇴 나이" 툴팁: 은퇴 늦추면 왜 좋은지 상세 설명
- "월 수령액 (65세 기준)" 툴팁: 친절한 안내 문구
- "연 인출 금액" 툴팁: "매년 고르게 나눠 쓰기" 용어 반영
- 결과 패널 합산 생활비에 "입력 생활비 + 건보료" 분리 설명 툴팁 추가
- 생활비 변경 툴팁: "작은 절감" → "이 금액이 매년 쌓이면" 수정

### 결과 패널 (우측)
- 연금 컬럼 동적 표시: 퇴직연금/개인연금 배열이 비어있으면 0으로 처리 (기존 호환 필드 참조 제거)
- 국민연금 수령액 0이면 플랜 요약에서 국민연금 행 숨김
- 실패 시 달성된 계좌는 권장 잔액 목록에서 숨김
- localStorage에서 결과 자동 복원 제거 — 계산하기 눌러야만 결과 표시

### 로그인 페이지
- 배경: Unsplash 이미지 10장 롤링 (12초 간격, 리조트/해변/자연 테마)
- 캐릭터 위치/크기 유지

### 데모 시스템 (AutoDemo.tsx)
- 헤더 위 다크 배너: "처음이신가요? ▶ 데모 보기 — 1분 안에 사용법을 알려드려요"
- localStorage 기반 닫기 (캐시 삭제 시 재표시)
- box-shadow 방식 spotlight + CSS transition으로 부드러운 이동
- 메시지 fade in/out 전환
- 35세 프리셋 시나리오 전체 흐름: 선택→계산→실패→전략적용→인풋확인→성공→앞당기기→채우기→재계산
- 변경된 인풋만 정확히 spotlight (findInputRow, findAppliedInputs)
- 결과 패널 옆 메시지 (messagePosition: "right")
- 배속 조절 (0.5x / 1x / 1.5x / 2x)
- 기본 속도 1.5배 기준으로 delay 재조정
