## 1. UI/UX 개선

### 입력폼
- 페이지 타이틀: "은퇴 시뮬레이션" → "탈출지도"
- 부제: "Power J의 정밀 로직으로 분석하는 나의 자산 수명 예측"
- "초기화" → "내 정보 직접 입력하기" (PenLine 아이콘, 보더 없음, hover 시 연한 회색 배경)
- "예시 불러오기" → "연령별 탈출 시나리오" (파란 배경 드롭다운)
- "시뮬레이션 종료 나이" → "몇 세까지 계산할까요?"
- "시뮬레이션 실행" → "계산하기" (펄스 애니메이션, disabled 시 그림자 제거)
- "연금소진모드" → "매년 고르게 나눠 쓰기" / "필요할 때만 꺼내 쓰기" (토글)
- 프리셋 설명: 대화체로 변경 ("현재 35세 싱글이고, 은퇴는 55세를 하려고 해요...")
- 프리셋 설명 박스: 셀렉트 하단에 위쪽 화살표로 연결, 닫으면 완전히 사라짐
- 62세 프리셋: retirementStartAge 60 → 62, 설명 문구 수정

### 섹션별
- 1층 국민연금 설명: "국민연금공단에서 받는 공적연금이에요. 내 연금 예상액은 국민연금공단 홈페이지에서 확인하세요."
- 2층 퇴직연금: 유형 선택 버튼 → 다크그레이 셀렉트박스 (IRP/DC/DB)
- 3층 개인연금: 유형 선택 버튼 → 셀렉트박스 (연금저축/연금보험)
- 추가 자산: 유형 라벨 → 셀렉트박스 (ISA/해외직투/예적금 등)
- 기타 자산: 이름 입력 필드 제거
- 부채 & 비정기 지출: 🔻 아이콘 제거
- "은퇴 시점 / 적립식" 토글: pill 스타일 (회색 배경 + 흰색 활성 탭)
- 모든 섹션 추가 버튼: dashed 보더 영역 ("+ 퇴직연금 계좌 추가" 등)

### 삭제 버튼 통일
- 기본: 투명 배경 + 회색 휴지통 아이콘 (Trash2)
- hover: 연분홍 배경 + 빨간 아이콘
- 아이콘 색상: `#ccc` → `#aaa` (더 진하게)

### 카드 레이아웃
- 각 계좌 항목: 카드형 (bg-secondary + border + borderRadius 10)
- 카드 내부 인풋: 흰색 배경 (.asset-card input { background: #fff })
- 인출 설정 영역도 asset-card 클래스 적용
- 비활성 인풋 (매년 고르게 나눠 쓰기 ON 시 연 인출 금액): disabled + 회색 배경

### 툴팁
- 스타일: 검정 배경 → 흰색 배경 + 그림자 + 뾰족 화살표
- 위치: 기본 위쪽, 공간 부족 시 아래쪽 자동 전환
- 희망 은퇴 나이 툴팁: 은퇴 늦추면 왜 유리한지 상세 설명
- 월 수령액 툴팁: "65세에 정상 수령할 때 받는 월 금액을 입력해 주세요..."
- 연 인출 금액 툴팁: "'매년 고르게 나눠 쓰기'를 끄면 매년 이 금액만큼 인출해요..."
- 결과 패널 합산 생활비: "입력한 생활비 X원 + 건보료 약 Y원이 합산된 금액이에요" 툴팁

### Hover 스타일 전체 추가
- 삭제 버튼: 연분홍 + 빨간 아이콘
- 추가 영역 (dashed): 파란 배경 + 파란 보더
- 계산하기 버튼: brightness + 위로 이동 + 그림자
- 초기화 버튼: 연한 회색 배경
- 셀렉트: 보더 파란색
- 프리셋 드롭다운: brightness 변경

### PC 정렬
- 기본 정보 라벨: 고정 너비 140px (인풋 시작점 정렬)
- 시작년도 변경 시 희망은퇴나이 자동 계산

## 2. 결과 패널 (우측)

### 용어 변경
- "Action Plan" → 코드 내부에서만 사용, UI에서는 "결과 패널" / "전략 반영하기"
- "J의 조언" → MD에서 제거 (코드에서는 advice/actionTip 유지)

### 실패 카드
- 전략 반영 시 자동 재계산 (버튼 1번 클릭으로 완료)
- 전략 강도: 총 적자 합계 기반으로 역산하여 성공 보장
- 달성된 계좌는 실패 시 숨김 (미달성만 표시)

### 성공 카드
- "내 은퇴 플랜 요약" 표시 (은퇴 시작, 월 생활비, 국민연금, 연금 인출, 시뮬 기간, 부채)
- 국민연금 수령액 0이면 플랜 요약에서 국민연금 행 숨김
- "전략 반영하기" 대신 요약 + 핵심 포인트 표시

### 패널 애니메이션
- 열림/닫힘: width transition 0.25s ease
- 초기화 시 패널 자동 닫힘

### 전략 배지
- 입력폼에 전략 반영 시 teal(#00b1bb) 배경 인풋으로 표시

## 3. 계산 로직

### 적립식 계산 모드
- 추가자산 / 개인연금: "은퇴 시점" / "적립식" 토글
- 적립식: 현재 잔액 + 연 납입액 + 수익률 → 은퇴 시점 복리 FV 계산
- 복수 퇴직연금: retirementPensions 배열, 가중평균 수익률
- 복수 개인연금: personalPensions 배열, 가중평균 수익률

### 동적 컬럼
- 입력값 0인 컬럼 숨김 (연금 배열 비었으면 0으로 처리)
- 퇴직연금/개인연금 배열 비어있으면 기존 호환 필드 대신 0

### 상수 업데이트
- 분리과세 한도: 1,200만 → 1,500만원 (2024년 개정)
- 재산 기본공제: 1억 3,500만 → 1억원 (일괄)
- 해외주식 수익률: 6% 미만 시 Math.max(0, ...) 처리

### localStorage
- 결과 자동 복원 제거 (계산하기 버튼 눌러야 결과 표시)
- 입력값만 복원

## 6. 자동 데모 (AutoDemo.tsx)

### 데모 배너
- 헤더 위 고정 다크 띠: "처음이신가요? ▶ 데모 보기 — 1분 안에 사용법을 알려드려요"
- 닫기: localStorage에 저장, 캐시 삭제 시 복원
- 탈출지도 페이지에서만 표시

### 데모 흐름
1. 35세 싱글 프리셋 선택 (항목 spotlight 강조)
2. 파란 설명 박스 확인
3. 바로 계산하기 (버튼 spotlight)
4. 실패 결과 확인 (우측 패널 spotlight + 우측 메시지)
5. "제이의 전략으로 즉시 탈출하기" 클릭
6. 변경된 인풋 하나씩 spotlight (은퇴나이, 75세 이전 생활비, 75세 이후 생활비, 국민연금 개시나이, 수익률)
7. 성공 결과 확인
8. 은퇴 앞당기기 배너 클릭
9. 채우기 버튼 순차 클릭
10. 재계산
11. 최종 결과 확인

### UI
- 어두운 오버레이 + box-shadow spotlight (CSS transition으로 부드럽게 이동)
- 메시지: fade in/out 전환
- 우측 패널 설명 시 "right" 위치에 메시지
- 배속 조절: 0.5x, 1x, 1.5x, 2x (하단 pill 컨트롤)
- 기본 속도: 1x (기존 1.5x 기준으로 delay 재조정)
