# 탈출로드맵 은퇴계산기 - 테스트 결과 체크리스트

> 판정 기준: 시뮬레이션 결과에서 **"파이어족 성공 ✅"** 또는 **"파이어족 실패 ❌ + 실패 나이"** 확인
>
> - **FIRE 성공**: 은퇴~시뮬레이션 종료까지 매년 남는금액 ≥ 0
> - **FIRE 실패**: 특정 나이에서 연간 남는금액 < 0 (= 자산 고갈)

## 상세 확인 항목

### TC-01: 사회초년생 최소입력
- [ ] 시뮬레이션 정상 실행됨
- [ ] 55~65세 구간: 소득 0원, 자산 없어 즉시 적자
- [ ] 65세 이후 국민연금 50만원 수령 시작 확인
- [ ] 생활비 200만원 vs 연금 50만원 → 매년 약 1,800만원 적자
- [ ] 예상 실패 나이: **55세** (첫 해부터 적자)

### TC-02: 30대 직장인 기본형
- [ ] DC 3,000만원 + 연금저축 2,000만원 → 은퇴 시 복리 성장 금액 확인
- [ ] 55~65세 공백기: 퇴직연금/개인연금 인출로 버티는지 확인
- [ ] 65세 이후 국민연금 80만원 수령 반영
- [ ] 예상 실패 나이: **60~65세 사이** (자산 소진 후)

### TC-03: 40대 맞벌이 고자산
- [ ] DB(수익률 0%) + IRP(5%) 가중평균 계산 정상
- [ ] ISA 5,000만원 + 해외직투 1억 인출 순서 확인 (예적금→ISA→해외 순)
- [ ] 부채 월 100만원 상환이 60세까지 반영
- [ ] 물가상승률 2.5% 적용 시 생활비 증가 패턴 확인
- [ ] 해외직투 배당 수령 + 필요시 매도 확인

### TC-04: 50대 은퇴 임박 (조기수령 63세)
- [ ] 국민연금 63세 시작 → 조기수령 감액률 적용 확인 (정상 대비 -12%)
- [ ] 55~63세 구간: 예적금 2억 + 퇴직연금 인출로 버틸 수 있는지
- [ ] 세금 계산: 55세 미만 16.5% vs 55~69세 5.5% 적용 확인

### TC-05: 국민연금 연기수령 70세
- [ ] 70세 수령 시 +36% 증액 반영 → 월 130만 × 1.36 = 약 177만원
- [ ] 55~70세 15년간 ISA 1억 + IRP 1억으로 생활비 충당 가능한지
- [ ] 연금보험 수익률 4% 반영 확인
- [ ] 예상: 연기수령 이득 vs 15년 공백기 자산소진 비교

### TC-06: 60대 이미 은퇴
- [ ] 현재나이=은퇴나이(62세) → 시뮬레이션 즉시 시작
- [ ] 시작년도 자동 계산 정상 (현재 연도)
- [ ] 62~65세: IRP+연금저축+예적금으로 3년 버티기
- [ ] 65세 이후: 국민연금 90만원 추가 수령으로 안정화

### TC-07: 주택연금 활용
- [ ] 주택연금 60세 시작, 월 100만원 수령 반영
- [ ] 58~60세 부채 상환(월 50만) + 생활비 → 자산 급감
- [ ] 60세 이후 주택연금 수령 시작으로 현금흐름 개선
- [ ] 부채 상환 63세 종료 후 월 지출 감소 반영

### TC-08: 비정기 지출 다수
- [ ] 57세 자녀결혼 5,000만원 반영
- [ ] 58세 차량교체 4,000만원 반영
- [ ] 60세 자녀결혼 5,000만원 반영
- [ ] 70세 의료비 3,000만원 반영
- [ ] 비정기 지출 총 1.7억이 해당 연도에 일시 차감
- [ ] 연속 대형 지출(57~60세)이 자산 고갈 가속화하는지 확인

### TC-09: 적립식 모드
- [ ] DC 적립식: 500만 + 연 300만 × 25년 × 5% 복리 → 은퇴 시 약 1.6억
- [ ] 연금저축 적립식: 200만 + 연 600만 × 25년 × 5% 복리 → 약 2.9억
- [ ] ISA 적립식: 100만 + 연 400만 × 25년 × 5% 복리 → 약 1.95억 (한도 주의: 1인 1억 캡)
- [ ] ISA 한도 초과 시 경고 표시 또는 1억 캡 적용 확인
- [ ] 복리 계산 공식 정확성: FV = PV×(1+r)^n + PMT×((1+r)^n-1)/r

### TC-10: 물가 5% 스트레스
- [ ] 물가 5% 적용 시 55세 생활비 300만 → 65세 약 489만 → 75세 약 796만
- [ ] 수익률 5~7%와 물가 5% → 실질 수익률 0~2%
- [ ] 자산 소진 속도가 기본(2%) 대비 급격히 빨라지는지 확인
- [ ] 예상 실패 나이: 기본 케이스보다 **5~10년 일찍**

### TC-11: 최소 생활비 100만원
- [ ] 입력 시 최소값(100만원) 경고/허용 확인
- [ ] 75세 이후도 100만원 직접 입력
- [ ] DC 5,000만 → 55세 시 약 7,400만(4%×10년 복리)
- [ ] 55~62세: DC에서 연 1,200만+건보료 인출 → 62세쯤 고갈
- [ ] 62~65세: 자산 0, 연금 0 → 3년간 완전 적자
- [ ] 65세 이후: 국민연금 80만(연960만) vs 생활비+물가(연~1,450만) → 연 490만 적자 지속

### TC-12: ISA 한도 1억
- [ ] ISA 1억 입력 시 경고 또는 정상 처리
- [ ] ISA 인출 순서: 부족분 발생 시에만 인출
- [ ] ISA 수익률 5% 반영으로 잔액 성장 확인

### TC-13: 부채 과다
- [ ] 월 부채 200만 + 월 생활비 250만 = 월 450만 지출
- [ ] 55~65세: 자산 5,000만 + 퇴직연금 3,000만 → 연 5,400만 필요
- [ ] 8,000만원으로 약 1.5년 → 즉시 적자
- [ ] 70세 부채 상환 완료 후 지출 급감 → 그때까지 버틸 수 있는지
- [ ] 예상 실패 나이: **56~57세**

### TC-14: 생명보험+개인연금 복합
- [ ] 연금저축(5%) + 연금보험(3%) 각각 별도 수익률 적용
- [ ] 생명보험연금 60세 시작, 월 80만원 수령
- [ ] 55~60세: IRP+연금저축+연금보험 인출
- [ ] 60세 이후: 생명보험 80만 추가 → 현금흐름 개선
- [ ] 65세 이후: 국민연금 100만 추가 → 안정화

### TC-15: 100세 초장기
- [ ] 50세~100세 = 50년간 시뮬레이션 정상 실행
- [ ] 물가 2% × 50년 → 75세 이후 생활비 약 2배
- [ ] 100세까지 데이터 테이블/차트 정상 출력
- [ ] 마지막 행(100세) 잔액 확인

### TC-16: FIRE족 40세 은퇴
- [ ] 40~65세 = 25년 공백기, 해외직투 3억 + ISA 1억으로 버티기
- [ ] 해외배당: 3억×6%=1,800만(세전) → 세후 1,522만 (15.4% 원천징수)
- [ ] 생활비 연 2,400만 vs 세후배당 1,522만 → 연 878만 적자
- [ ] ISA 인출로 부족분 보전 → ISA 소진 시점 확인
- [ ] 65세 이후 국민연금 40만원 추가해도 적자 지속

### TC-17: 다중 퇴직연금 3개
- [ ] DB(0%) + DC(4%) + IRP(5%) 세 계좌 모두 표시
- [ ] DB 수익률 0% 고정 확인 (1.5억 그대로)
- [ ] 총 퇴직연금 잔액: 1.5억 + 5,000만 + 3,000만 = 2.3억
- [ ] 인출 시 세금 계산 (연령별 세율 적용)

### TC-18: 무자산 연금만 의존
- [ ] 55~60세: 자산 0, 연금 0 → 즉시 적자
- [ ] 60세 조기수령 시 감액: -30% (5년 조기) → 월 56만원
- [ ] 생활비 200만 vs 연금 56만 → 연 1,728만 적자
- [ ] 예상 실패 나이: **55세** (첫 해부터)

### TC-19: 물가 0%
- [ ] 생활비가 35년간 변동 없음 (300만원 유지)
- [ ] 수익률만 5~7% 적용 → 자산 계속 성장
- [ ] TC-10(물가5%)과 비교 시 결과 큰 차이
- [ ] 예상: 모든 자산이 충분히 성장해 **확실히 성공**

### TC-21: 프리랜서 추가자산만
- [ ] 국민연금 0원 입력 시 정상 처리
- [ ] 퇴직연금/개인연금 없음 상태에서 시뮬레이션 정상 실행
- [ ] 해외배당: 4억×6%=2,400만(세전) → 세후 2,030만 (15.4% 원천징수)
- [ ] 생활비 연 3,000만 vs 세후배당 2,030만 → 연 970만 적자
- [ ] 부족분은 ISA → 예적금 순으로 인출
- [ ] 65세 이후에도 국민연금 0원 → 추가자산에만 계속 의존
- [ ] 75세 이후 생활비 175만 감소 시 적자폭 축소, 흑자 전환 여부

### TC-22: 상속자산 부동산+현금
- [ ] 국민연금 0원, 퇴직연금 없음, 개인연금 없음 정상 처리
- [ ] 50~55세: 예적금 3억에서 생활비(300만)+부채상환(80만) = 월 380만원 인출
- [ ] 55세 주택연금 시작 → 월 150만원 수령으로 부담 경감
- [ ] 60세 부채 상환 완료 → 월 지출 300만원으로 감소
- [ ] 예적금 소진 시점 확인 (주택연금 150만 vs 생활비 300만 → 월 150만 부족)
- [ ] 75세 이후 생활비 210만원 vs 주택연금 150만원 → 여전히 월 60만 부족

### TC-20: 모든 항목 풀입력
- [ ] 7개 섹션 모든 입력 항목 정상 반영
- [ ] DB + IRP 퇴직연금 2개 계좌 처리
- [ ] 연금저축 + 연금보험 개인연금 2개 계좌 처리
- [ ] ISA + 해외직투 + 예적금 + 주택연금 + 생명보험 5개 자산 처리
- [ ] 부채 2억, 월 150만 상환, 65세 종료 반영
- [ ] 비정기 지출 3건 (58세, 60세, 75세) 반영
- [ ] 물가 3% 적용
- [ ] 국민연금 66세 연기 → +7.2% 증액
- [ ] 인출 순서: 예적금 → ISA → 해외주식 정상 작동
- [ ] 시뮬레이션 테이블에 모든 항목 컬럼 표시
- [ ] CSV/Excel/JSON/텍스트 내보내기 정상 동작
