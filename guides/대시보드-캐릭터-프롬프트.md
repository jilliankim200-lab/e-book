## 초기 세팅 (대화 시작 시 1회만 입력)

> 캐릭터 이미지를 첨부한 후 아래 프롬프트를 입력하세요.

### 세팅: 캐릭터 등록

```
앞으로 이 캐릭터의 3D 스타일을 '나노바나나 스타일'이라고 부를게.
이후 모든 이미지 생성은 '나노바나나 스타일'을 참고해줘.

나노바나나 스타일 규칙:
- 3D 스타일로 유지할 것
- 반드시 첨부한 이 캐릭터로만 구성할 것. 다른 캐릭터로 대체하지 마
- 캐릭터의 얼굴, 머리 모양, 안경, 체형 비율, 눈코입 스타일을 정확히 유지해줘. 절대로 변형시키지 마
- 눈은 반드시 흰 흰자위 안에 큰 갈색 동그란 눈동자가 있는 원본 스타일 그대로 유지할 것. 흰자 없이 눈동자만 있거나, 눈 모양을 변형하지 마
- 소품, 배경, 포즈, 의상도 상황에 맞게 연출해줘
- 화질 저하 없이 초고화질로 생성하고 선명하게 유지할 것
- 배경은 반드시 완전한 투명(transparent background)으로 처리할 것. 배경색을 넣지 마
- 배경에 한글 텍스트가 필요한 경우, 한글 대신 영어로 표기하거나 텍스트 없이 심볼/로고만 사용할 것
```

## 01. 랜딩 페이지 — 히어로 섹션

### 프롬프트 2: 대시보드를 자랑하는 모습
해당 캐릭터가 [투명한 배경(transparent background) 위에, 양손을 활짝 벌려 카메라를 향해 넓은 홀로그램 대시보드 화면을 보여주고 있는] 모습 만들어줘. 홀로그램 화면에 컬러풀한 막대그래프, 도넛차트, 숫자 카드, 지도 히트맵이 예쁘게 배치되어 떠 있고, 캐릭터는 "짠!" 하는 표정으로 밝게 웃고 있어. 캐릭터 전신이 보이도록 할 것. 나노바나나 스타일로 생성해줘.

### 프롬프트 3: 드래그 앤 드롭 제스처
해당 캐릭터가 [투명한 배경(transparent background) 위에, 한 손으로 초록색 엑셀 파일 아이콘을 집어서 위쪽의 빛나는 업로드 영역으로 올려놓으려는 제스처를 하고 있는] 모습 만들어줘. 업로드 영역에서 환한 빛이 나오고, 캐릭터는 집중하면서도 기대에 찬 표정. 캐릭터 전신이 보이도록 할 것. 나노바나나 스타일로 생성해줘.

```
이 캐릭터의 3D 애니메이션 영상을 만들어줘.
둥근 안경 쓴 갈색 단발머리 소녀 캐릭터가 깔끔한 캐주얼 차림으로 서 있어.

동작: 캐릭터가 한 손에 들고 있는 초록색 엑셀 파일을 위로 가볍게 던져 올리면, 파일이 공중에서 반짝이며 화려한 대시보드(막대그래프, 파이차트, 꺾은선 그래프 조각들)로 변환되어 흩어진다. 캐릭터가 양손을 활짝 벌려 대시보드 조각들을 받으며 환하게 미소 짓는다. 차트 조각들이 다시 모여 파일로 돌아오며 루프.

배경: 완전한 단색 배경(흰색 #FFFFFF). 배경에 오브젝트나 풍경 없음.
카메라: 정면 고정. 줌인/줌아웃 없음. 전신 중앙 고정.
음성: 없음. 무음.
10초 이내. 심리스 루프로 제작할 것.
```
