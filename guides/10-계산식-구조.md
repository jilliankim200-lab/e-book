# 탈출지도 계산식 구조

> **최종 업데이트**: 2026-03-27

이 문서는 탈출지도(은퇴 시뮬레이션)의 연도별 결과 테이블에 사용되는 모든 계산식을 정리한 것입니다.

---

## 시뮬레이션 흐름 (매년 반복)

```
1단계: 생활비 계산 (물가상승률 복리 적용)
2단계: 고정 수입 계산 (국민연금, 주택연금, 생명보험연금)
3단계: 변동 수입 계산 (해외배당, 연금 인출, ISA 인출, 해외주식 매도)
4단계: 건강보험료 계산 (개인연금 기반)
5단계: 부채 상환 + 비정기 지출
6단계: 총계 (총소득, 총지출, 남는 금액)
7단계: 부족분 보전 (예적금 -> ISA -> 해외주식 순)
8단계: 잔액에 수익률 적용 (연말 기준)
```

---

## 1. 연 생활비

```
월 생활비 x 12 x (1 + 물가상승률/100) ^ 연차
```

- 75세 미만: 75세 이전 월 생활비 적용
- 75세 이상: 75세 이후 월 생활비 적용
- 연차 = 현재나이 - 은퇴시작나이 (0부터 시작)

---

## 2. 국민연금 (세후)

```
기준연금액 x 조기/연기 조정계수 x (1 + 물가상승률/100) ^ 경과년수 x (1 - 나이별 세율)
```

### 조기/연기 조정계수

| 수령 시기 | 조정 방식 |
|----------|----------|
| 65세 미만 (조기) | 1 - (65 - 수령시작나이) x 0.06, 최대 5년(30% 감액) |
| 65세 (정상) | 1.0 (조정 없음) |
| 65세 초과 (연기) | 1 + (수령시작나이 - 65) x 0.072, 최대 5년(36% 증액) |

### 나이별 연금소득세율

| 나이 | 세율 |
|------|------|
| 55세 미만 | 16.5% (기타소득세) |
| 55~69세 | 5.5% |
| 70~79세 | 4.4% |
| 80세 이상 | 3.3% |

- 경과년수 = 현재나이 - 수령시작나이 (첫 해는 0 = 물가상승 미적용)

---

## 3. 주택연금

```
월 수령액 x 12
```

- 설정한 개시 나이부터 매년 동일 금액 수령
- 비과세 소득 (세금 없음)

---

## 4. 생명보험연금

```
연 수령액 (고정)
```

- 설정한 개시 나이부터 매년 동일 금액 수령

---

## 5. 해외직투 배당 (세후)

```
해외주식 잔액 x 배당률(6%) x (1 - 원천징수 15.4%)
```

- 배당소득세 15.4% 공제 후 수령

---

## 6. 개인연금 인출 (세후)

### 일반 모드

```
인출액 = MIN(필요금액/0.945, 설정 인출액, 연금잔액)
```

- 필요금액 = 생활비 + 건보료 - 고정수입 - 해외배당 (부족분만 인출)

### 연금소진모드 (PMT 공식)

```
인출액 = 잔액 x (r x (1+r)^n) / ((1+r)^n - 1)
```

- r = 연금수익률
- n = 남은 년수 (시뮬레이션 종료나이 - 현재나이 + 1)
- 시뮬레이션 종료 시 잔액이 정확히 0원이 되도록 균등 인출

### 세금 계산 (공통)

```
분리과세 한도(1,500만원) 이내: 인출액 x 나이별 세율
한도 초과분: 초과액 x 초과세율(사용자 설정)
세후 = (한도이내 - 세금) + (초과분 - 세금)
```

---

## 7. ISA 인출

### 일반 모드

```
ISA 인출 = MIN(부족분, ISA 잔액)
```

- 부족분 = 생활비 + 건보료 - 고정수입 - 해외배당 - 연금세후
- 다른 소득으로 충당 후 부족한 금액만 ISA에서 인출

> **참고**: 해외주식 매도 시 양도소득세(22%, 연 250만원 비과세)는 현재 미반영됩니다. 실제보다 유리하게 계산될 수 있습니다.

### 연금소진모드

```
ISA 인출 = ISA잔액 x (r x (1+r)^n) / ((1+r)^n - 1)
```

- r = ISA 수익률, n = 남은 년수
- ISA도 PMT 공식으로 균등 소진

---

## 8. 해외주식 매도

### 일반 모드

```
해외주식 매도 = MIN(부족분, 해외주식 잔액)
```

- 해외배당 + 연금 + ISA로도 부족한 경우에만 매도

### 연금소진모드

```
해외주식 매도 = 해외주식잔액 x (r x (1+r)^n) / ((1+r)^n - 1)
```

- r = 해외주식수익률 - 배당률(6%) = 성장분 수익률만 적용
- 배당은 별도 자동 수령되므로 성장분만 소진 대상

---

## 9. 예적금 인출

```
예적금 인출 = 부족분 보전 시 예적금에서 먼저 인출
```

- 총소득 - 총지출이 마이너스일 때 예적금 -> ISA -> 해외주식 순으로 보전

---

## 10. 건강보험료 (지역가입자)

```
건보료 = (소득점수 + 재산점수) x 218.8원 x 12개월
```

### 소득점수

```
소득점수 = 개인연금 세후 x 0.002 / 1,000
```

- 건보 반영 소득: **개인연금만**
- 미반영: 국민연금, 주택연금, ISA, 해외배당

### 재산점수

```
재산점수 = MAX(0, 공시가격 - 1억원) x 0.0000005
```

- 재산 기본공제: 1억원 (일괄 적용)
- 주택은 시가가 아닌 **공시가격(시가의 60~70%)** 기준으로 산정

---

## 11. 부채 상환

```
초기 부채잔액 = 은퇴시작나이 > 상환종료나이 ? 0 : 입력 부채금액
부채상환 = MIN(월 상환액 x 12, 부채잔액)
```

- 은퇴 시작 전에 상환이 완료되는 경우 부채잔액 0으로 시작
- 상환 완료 나이까지만 적용
- 부채잔액 = 이전잔액 - 상환액

---

## 12. 비정기 지출

```
비정기지출 = 해당 나이에 설정한 금액의 합계
```

- 자녀 결혼, 의료비 등 특정 나이에 발생하는 일회성 지출

---

## 13. 총소득

```
총소득 = 연금세후 + 국민연금 + 주택연금 + 생명보험연금
       + ISA인출 + 해외배당 + 해외주식매도 + 예적금인출
```

---

## 14. 총지출

```
총지출 = 연 생활비 + 건보료 + 부채상환 + 비정기지출
```

---

## 15. 남는 금액

```
남는 금액 = 총소득 - 총지출 - 추가차감
```

- 추가차감: 국민연금 수령 시작 나이부터 연 500만원 (국민연금 수령 시 발생할 수 있는 추가 세금 및 비정기 의료비를 위한 안전 마진)

---

## 16. 부족분 보전 순서

총소득 - 총지출이 마이너스일 때:

```
1순위: 예적금에서 인출
2순위: ISA에서 추가 인출
3순위: 해외주식 추가 매도
```

---

## 17. 잔액 계산 (연말, 수익률 적용)

모든 인출/매도가 완료된 후 잔액에 수익률 복리 적용:

```
ISA 잔액 = (인출 후 잔액) x (1 + ISA 수익률)
해외주식 잔액 = (매도 후 잔액) x (1 + 해외주식수익률 - 배당률6%)
연금 잔액 = (인출 후 잔액) x (1 + 연금 수익률)
예적금 잔액 = (인출 후 잔액) x (1 + 예적금 수익률)
```

- 해외주식은 배당 6%가 별도 수령되므로 잔액에는 성장분(수익률-6%)만 적용

---

## 18. FIRE 성공/실패 판정

```
실패 조건: 시뮬레이션 기간 중 남는 금액이 마이너스인 연도 발생
```

- 모든 자산(예적금, ISA, 해외주식)에서 보전해도 부족하면 실패
- 실패 시 해당 나이와 부족 원인을 분석하여 표시
