# 탈출로드맵 시스템 체크리스트

> **목적**: 계산식 정확성 + 화면 UI를 체계적으로 검증하기 위한 체크리스트
> **실행 방법**: "체크리스트 실행해"라고 요청하면 이 문서를 기준으로 전체 검증 수행
> **최종 업데이트**: 2026-03-28

---

## 1. 계산식 정확성 검증

### 1-1. 생활비 계산
- [ ] 75세 이전/이후 생활비가 각각 적용되는지
- [ ] 물가상승률 복리가 매년 올바르게 적용되는지 (연차 = 현재나이 - 은퇴시작나이)
- [ ] 물가상승률 0%일 때 생활비가 변하지 않는지

### 1-2. 국민연금
- [ ] 조기수령(60~64세): 1년당 6% 감액이 정확한지
- [ ] 정상수령(65세): 조정 없음
- [ ] 연기수령(66~70세): 1년당 7.2% 증액이 정확한지
- [ ] 나이별 연금소득세율 적용 (55~69세: 5.5%, 70~79세: 4.4%, 80세+: 3.3%)
- [ ] 물가상승률이 연금에 반영되는지

### 1-3. 개인연금 인출
- [ ] 일반 모드: 부족분만 인출하는지
- [ ] 소진 모드(PMT): 균등 인출 공식이 정확한지 (r*(1+r)^n / ((1+r)^n - 1))
- [ ] 분리과세 한도(1,500만원) 적용이 맞는지
- [ ] 한도 초과분에 초과세율이 적용되는지

### 1-4. ISA/해외직투/예적금
- [ ] ISA 인출 순서: 연금 다음에 ISA에서 인출하는지
- [ ] 해외배당: 잔액 × 배당률(6%) × (1 - 15.4%) 공식이 맞는지
- [ ] 부족분 보전 순서: 예적금 → ISA → 해외주식 순인지
- [ ] 각 계좌 잔액에 수익률이 연말 기준으로 복리 적용되는지

### 1-5. 부채
- [ ] 은퇴시작나이 > 부채상환종료나이 → debtBalance = 0 으로 시작하는지
- [ ] 상환 종료 나이 이후에는 상환이 멈추는지
- [ ] 월 상환액 × 12가 연간 상환액으로 적용되는지

### 1-6. 건강보험료
- [ ] 개인연금 기반 건보료가 계산되는지
- [ ] 국민연금 수령 나이부터 추가 차감(500만원)이 적용되는지

### 1-7. 비정기 지출
- [ ] 설정한 나이에 정확히 일시 지출로 반영되는지
- [ ] 여러 개 등록 시 각각 올바른 나이에 반영되는지

## 3. 우측 패널 (ActionPlan) 검증

### 3-1. 실패 시
- [ ] 실패 카드: 캐릭터 + 메시지 + 부족 금액/어려운 시점 표시
- [ ] 계좌별 권장 잔액이 실패 시 표시되는지
- [ ] 채우기 버튼 동작: 부족분만큼 해당 계좌에 반영
- [ ] 게이지 드래그: 사용자가 금액 조절 가능
- [ ] 채우기 후 "즉시 탈출하기" 버튼 숨김
- [ ] "채운 금액으로 다시 계산하기": 전략 카드 변경사항 + 채운 금액 함께 반영
- [ ] 전략 카드: 제목 + 설명 + 실행 팁 + 상세 (J의 조언 숨김 상태)

### 3-2. 성공 시
- [ ] 성공 카드: 녹색 배경 + 캐릭터 + 서술형 메시지
- [ ] "90세까지 남는 금액/가장 어려운 시점" 영역 숨김
- [ ] 내 은퇴 플랜 요약: 각 항목에 툴팁 (동적 - 변경된 항목은 이유 설명)
- [ ] 은퇴 시작: "65세 (63세까지 가능)" 형태로 여유분 표시
- [ ] 월 생활비: "150만원 (180만원까지 가능)" 형태로 여유분 표시
- [ ] 부채: 은퇴 전 상환 완료 시 "은퇴 전 상환 완료" 표시
- [ ] 핵심 포인트: 가장 어려운 시점 + 국민연금 공백 + 잔여 자산(totalAssetEnd 숨김 상태)

### 3-3. 전략 계산
- [ ] 생활비 증가 역산: 총 잉여 50% × 복리 승수 반영
- [ ] 은퇴 앞당기기 역산: 복리 기회비용 포함, 최대 5년
- [ ] 은퇴 앞당기기/생활비 증가가 버튼이 아닌 정보 텍스트로 표시
