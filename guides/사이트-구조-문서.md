## 프로젝트 개요

## 핵심 컴포넌트

### 은퇴계산기.tsx (CashFlow)
- **역할**: 메인 시뮬레이션 엔진 + 입력 폼 + 결과 테이블
- **주요 기능**:
  - 유형 선택 프리셋 (35세 싱글 ~ 62세 은퇴자, 6종)
  - 기본 정보, 생활비, 국민연금, 퇴직연금, 개인연금, 추가자산, 부채, 비정기지출 입력
  - calculateSimulation() 함수로 연도별 현금흐름 계산
  - 결과 테이블 + CSV/Excel 다운로드
  - 생활비 계산 도우미 (팝업)

### ActionPlan.tsx
- **역할**: 우측 패널 - 성공/실패 판정 + 전략 제안
- **주요 기능**:
  - 성공 시: 녹색 카드 + 서술형 요약 + 플랜 요약 + 핵심 포인트
  - 실패 시: 빨간 카드 + 부족 금액/어려운 시점 + 전략 카드 제공
  - 전략 카드: J의 조언 + 실행 팁 + 상세 정보
  - "제이의 전략으로 즉시 탈출하기" 버튼 → 좌측 입력값 자동 수정 + 재계산

### App.tsx
- **역할**: 전체 레이아웃, 사이드바, 라우팅, 테마(다크/라이트), 폰트 크기 조절
- **레이아웃**: 좌측 사이드바 + 중앙 콘텐츠 + 우측 ActionPanel (3단 구조)

## 인증 체계

- **PasswordGate.tsx**: PC 로그인 (배경 이미지 롤링 + 캐릭터 동영상)
- **PasswordGate2.tsx**: 모바일 로그인 (단색 배경 + 캐릭터)
- **인증 방식**: 비밀번호 해시 비교 → sessionStorage 저장
- **권한**: admin / user 2단계

---

## 캐릭터 시스템

- **캐릭터명**: 제이 (나노바나나 스타일)
- **사용 위치**:
  - 로그인 페이지: 망원경 든 캐릭터 (동영상)
  - 웰컴 페이지: 망원경 탐색 캐릭터 (동영상)
  - 저자의 글: 출근길, 엑셀 작업, 완성 뿌듯함 (이미지)
  - 사례 페이지: 김영희 캐릭터 (이미지)
  - ActionPlan: 성공/실패 캐릭터 (이미지)
  - 유형 선택 팝업: 안내 캐릭터 (이미지)
- **프롬프트 가이드**: guides/캐릭터-프롬프트-가이드.md

## 빌드 및 배포

- **빌드 도구**: Vite
- **난독화**: vite-plugin-obfuscator (빌드 시 자동 적용)
  - controlFlowFlattening, deadCodeInjection, stringArray 등 활성화
  - debugProtection, disableConsoleOutput 활성화
- **빌드 명령**: `npm run build`
- **출력 폴더**: dist/
