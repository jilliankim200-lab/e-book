   탈출로드맵 설치 안내
═══════════════════════════════════════

▶ 설치 방법
   install.bat 파일을 더블클릭하세요.
   모든 것이 자동으로 설치됩니다.
   (인터넷 연결 필요, 약 3~5분 소요)

▶ 설치되는 항목
   - Node.js (서버 실행 도구)
   - Git (코드 관리 도구)
   - 탈출로드맵 프로젝트
   - 바탕화면 바로가기

▶ 설치 후 사용법
   바탕화면의 "탈출로드맵" 아이콘을
   더블클릭하면 자동으로 실행됩니다.

▶ 기타 파일
   update.bat  - 최신 버전으로 업데이트
   deploy.bat  - 프로덕션 사이트에 배포

▶ 문제 발생 시
   - install.bat을 "관리자 권한으로 실행"
   - 백신 프로그램이 차단할 수 있음
     → "허용" 선택 후 다시 실행
